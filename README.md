### 将网页保存为pdf或者图片
##### 因为一些文章会被作者或者平台强制删除，此项目旨在将还未删除的内容保存为image or pdf。

#### 保存方式
- [selenium+chrome](https://github.com/broholens/full-page-screenshot/blob/master/capture_screen_by_selenium.py)
- [imgkit+wkhtmltopdf](https://github.com/broholens/full-page-screenshot/blob/master/capture_screen_to_img.py)
- [pdfkit+wkhtmltopdf](https://github.com/broholens/full-page-screenshot/blob/master/capture_screen_to_pdf.py)
